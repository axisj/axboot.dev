# 변경 이력

Axboot API의 주요 변경 사항은 다음과 같습니다:

## 버전: v1.1.0 (예정)
- 날짜: 예정
- 예정된 변경 사항:
- 알림(Notification) API 추가
- 사용자 권한 관리 기능 개선
- 파일 업로드 크기 제한 설정 추가


## 버전: v1.0.0
- 날짜: 2024-12-01
- 주요 변경 사항:
- 초기 API 릴리스
- 사용자 관리 엔드포인트 (/auth, /users) 구현
- 메뉴 관리 엔드포인트 (/menus) 구현
- 파일 업로드 및 다운로드 엔드포인트 (/files) 추가
- 시스템 설정 엔드포인트 (/settings) 추가
