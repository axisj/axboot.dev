# 오류 코드

API 요청이 실패할 경우, 서버는 표준 HTTP 상태 코드와 함께 JSON 형식의 오류 응답을 반환합니다. 

아래는 주요 오류 코드의 설명입니다:

| HTTP 상태 코드 |	오류 코드 |	설명 |
|---|---|---|
| 400 | 	1000| 	잘못된 요청 (유효하지 않은 입력 데이터)|
| 401 | 	1001| 	인증 실패 (토큰 없음 또는 유효하지 않음)|
| 403 | 	1002| 	권한 없음|
| 404 | 	1003| 	요청한 리소스를 찾을 수 없음|
| 409 | 	1004| 	데이터 충돌 (중복 데이터 등)|
| 500 | 	1005| 	서버 내부 오류|
